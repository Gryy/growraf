Resize + Growing plugin bug

The above demonstrates a bug that occurs in flot charts using Resize and Growing plugin.

If the plot placeholder gets resized before the Grow animation ends, then the resulting plot will be invalid. The reason seems to be that, the Resize plugin redraws the plot when a 'resize' event occurs and as a result the Growing plugin continues the animation from there.
